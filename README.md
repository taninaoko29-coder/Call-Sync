# 📞 Call-Sync（コール・シンク）

コールセンターの「困った」を即座に解決し、対応の標準化を実現するオペレーター向けナレッジ共有ツールです。

## 💡 制作の背景と目的
コールセンターのアルバイト実務において、マニュアルにないイレギュラーな質問が発生した際、管理者（SV）への確認で「保留待機時間」が長引いてしまう課題がありました。
オペレーター同士が現場のノウハウを素早く検索・共有できるクローズドな環境を構築し、自己解決率の向上と業務の標準化（SDCAサイクルの「S」）を目指して開発しました。

## 🛠️ 使用技術・アピールポイント
* **PHP / MySQL**
* **1ファイルでのアジャイルな構成:** 現場への導入しやすさを考慮し、1つのPHPファイルで全機能が動くように設計しています。
* **セキュアな認証基盤:** 業務情報を守るため、社員番号とハッシュ化したパスワードによる強固なログイン機能を実装しました。
* **自作関数の活用:** 1ファイル構成の中でも、投稿取得処理などを独自の関数として切り分け、保守性を高めています。
